# input

## 简介
  输入框预设

## props
  1. label: String 当前Input选项需要展示的label
  3. value: String 默认值
  4. disabled: Boolean 当前Input是否置灰。
  5. search: Boolean 当前框展示search搜索样式。
  5. el-input的其他参数内容。

## event
  1. input: 当前value变化的时候会自动的触发当前的时间内容。
  2. change: 类似于input只是当input没有传递时候触发。

## API
  1. clear: 清空当前内容。
  2. input: 当前内容。
  3. disabled: 当前组件置灰
  4. abled: 当前组件可用
